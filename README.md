# Latar Belakang
Perkembangan teknologi dan industri game membuat aktivitas bermain game menjadi bagian dari kehidupan pelajar dan mahasiswa. Meskipun game dapat menjadi sarana hiburan dan interaksi sosial, intensitas bermain yang berlebihan dapat memengaruhi performa akademik, seperti waktu belajar, kualitas tidur, tingkat stres, dan nilai akademik.


Dataset Gaming Academic Performance digunakan untuk menganalisis hubungan antara kebiasaan bermain game dengan prestasi akademik mahasiswa. Melalui proses Exploratory Data Analysis (EDA), dapat diketahui pola, distribusi data, serta hubungan antarvariabel seperti durasi bermain game, jam belajar, tingkat kecanduan, dan nilai akademik. Analisis ini penting untuk memahami dampak aktivitas gaming terhadap pencapaian akademik dan membantu menemukan keseimbangan antara hiburan dan pendidikan.
